# 常用命令

| 命令名称                              | 作用 |
|-----------------------------------|--|
| git config --global user.name 用户名 | 设置用户签名 |
|git config --global user.email 邮箱|设置用户签名|
|git init|初始化本地库|
|git status|查看本地库的状态|
|git add 文件名|添加文件到暂存区|
|git commit -m "日志信息" 文件名字|提价本地库|
|git reflog|查看历史记录|
|git reset --hard 版本号|版本切换|
